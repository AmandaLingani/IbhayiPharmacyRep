The Pharmacist Subsystem is a web-based module designed to support pharmacist-specific workflows within a larger healthcare management system.
This repository contains only the Pharmacist Subsystem, which I independently designed and implemented.
Tech Stack
ASP.NET Core, C#, SQL Server, Entity Framework Core, HTML, CSS, JavaScript
Scope of Contribution
This project represents my personal contribution to a broader system.
My work focuses on:
Pharmacist-related workflows
Prescription handling and document uploads
Data management using Entity Framework Core
UI components related to pharmacist functionality
Other system modules are intentionally excluded from this repository.
Clone the repository
Create an appsettings.json file based on appsettings.example.json
Update configuration values as needed
Run the project using Visual Studio or dotnet run
Notes
This repository is part of my personal portfolio and reflects ongoing development as I continue to improve and expand the subsystem.
